# HomeBudgetTracker

## Opis

Aplikacja internetowa stworzona w technologii ASP.NET Core MVC służąca do monitorowania wydatków domowych.

## Technologie

- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- SQLite
- Bootstrap

## Funkcjonalności

- Dodawanie wydatków
- Edycja wydatków
- Usuwanie wydatków
- Lista wydatków
- Wyszukiwanie po kategorii
- Filtrowanie po dacie
- Dashboard z podsumowaniem wydatków
- Walidacja formularzy

## Uruchomienie

1. Sklonować repozytorium
2. W katalogu projektu wykonać:

dotnet restore

dotnet ef database update

dotnet run

3. Otworzyć aplikację w przeglądarce
